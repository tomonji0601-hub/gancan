# 土質試験サポートアプリ

GitHub Pages でそのまま公開できる、シンプルな静的Webアプリです。

## 機能

### 1. 含水比計算モード
- 入力
  - `ma`: 湿潤試料＋皿 (g)
  - `mb`: 乾燥試料＋皿 (g)
  - `mc`: 皿 (g)
- 出力
  - `w(%) = 100 × (ma - mb) / (mb - mc)`
- 追加機能
  - 試料名の入力
  - 保存時刻の自動取得
  - 履歴保存
  - CSV出力

### 2. 試料調整モード
- 入力
  - `w`: 調整後の含水比 (%)
  - `mi`: 湿潤試料の質量 (g)
  - `w1`: 湿潤試料の含水比 (%)`
- 出力
  - `mw`: 加える水の質量 (g)
- 使用式
  - `mw = mi / (1 + w1/100) × (1 + w/100) - mi`
- 備考
  - 1 mL = 1 g として扱う
- 追加機能
  - 試料名の入力
  - 保存時刻の自動取得
  - 履歴保存
  - CSV出力

## ファイル構成

- `index.html` : 画面本体
- `styles.css` : デザイン
- `app.js` : 計算・保存・CSV出力
- `manifest.json` : PWA設定
- `sw.js` : オフライン用キャッシュ

## GitHub Pages で公開する手順

1. GitHub で新しいリポジトリを作成
2. このファイル一式をアップロード
3. GitHub の `Settings` → `Pages` を開く
4. `Deploy from a branch` を選ぶ
5. Branch を `main` / `/root` に設定して保存
6. 数分後に公開URLが発行される

## ローカルで確認する方法

`index.html` をブラウザで開くだけでも動きます。

## データ保存について

- 履歴はブラウザの `localStorage` に保存されます
- 同じ端末・同じブラウザなら残ります
- 端末変更やブラウザデータ削除時には消えるため、必要に応じてCSV出力してください
